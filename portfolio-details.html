About the Cloud Resume Challenge

Inspired by the Cloud Resume Challenge created by Forrest Brazeal, I endeavored to complete the challenge using AWS resources. After some scope change, instead of creating just a cloud resume I created this site instead. I used many AWS services to complete this project, including S3, CloudFormation, AWS Lambda, Route 53, DynamoDB and other services. I also utilized a CI/CD pipeline using GitHub Actions.

About The Sleep App

With two other developers, I created a Sleep Tracking application. Users can log into individual accounts to track their their hours slept, sleep routine, quality of sleep, and a sleep diary. We implemented an option for users to delete their account and all user data. Although not completely implemented, we created a function to compare a user's average hours slept against reccomendations. The front end is built with React and Skeleton/Normalize. The back end is built with Django, PostgreSQL with log-in and encryption handled by bcrypt.

About the Bottle App

With another developer, I created the Bottle App. It is a online chat application with the ability to reply to and add tags to posts. We incorporated socket.io to enable user-created chat rooms that update in real time. We used React and Bootstrap to style the frontend, and utilized MongoDB, Node and Express for the backend.

Found Connections

  • Category: Web application
  • Client: Bootcamp Project
  • Project date: May, 2022
  • Repository: FoundConnections

About Found Connections

Found Connections is a website for an theoretical company that sells power and AV cables for legacy consoles. Users have the ability to schedule a meeting at a physical store to ensure the cable works correctly, using a CRT television for accuracy. In addition, there is an events page where users can find virtual and in-person gaming events like conferences and tournaments. This is a multi-page application using MongoDB, Express, and Node for the backend, while utilizing Javascript and the Bulma CSS framework on the frontend.
